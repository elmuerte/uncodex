HTML Template class_entry_struct*.html

These replacements can only be used in class_entry_struct*.html.
replacement replaced with 
%struct_name%
 the name of the structure 
%struct_extends%
 the name of the structure it extends 
%struct_extends_link%
 the name of the structure it extends including link 
%struct_extends_label%
 the word "extends" 
%struct_modifiers%
 modifiers of this structure 
%struct_srcline%
 the line number of the declaration 
%struct_comment%
 description of the structure 
%struct_properties_newline%
 struct fields seperated with a new line 
%struct_properties_break%
 struct fields seperated with a new HTML break 
%struct_properties_detail*%
 detailed structure field display, the format is used from the struct_properties_detail*.html template. This template uses the same replacements as class_entry_variable*.html. 
%struct_properties_comment*%
 same as above, except only the variables with a comment will be used. The template used is struct_properties_comment*.html, this also uses the same replacements as class_entry_variable*.html. 
%struct_declaration%
 Syntax highlighted declaration of the struct. This only contains the struct head, not it's body 
%has_comment_?%
 will be replaced with the HasCommentValue title when there's a comment 
%class_source%
 link to the class source file
