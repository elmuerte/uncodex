Welcome to UnCodeX

UnCodeX is a class browser and source document creator in one. It features improved functionality from UClasses and UnDox. UnCodeX can be used with any game based on the Unreal Engine.

Check the UnCodeX website on the UnrealWiki for the latest updates.

UnCodeX has been designed and written by Michiel 'El Muerte' Hendriks as a reaction to missing or incomplete functionality in UClasses and UnDox.

UnCodeX can be freely used by anyone. If you have any suggestions for possible improvements I'll be glad to hear them.

-- El Muerte
