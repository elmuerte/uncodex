HTML Templates

HTML Templates consist out of HTML formatting and replacements. Replacements are depended on the template file.

General replacements

These replacements can be used in any template.
replacement replaced with 
%default_title%
 DefaultTitle as set in the template.ini, defaults to "UnCodeX". This can be overwritten from the system config file. 
%create_time%
 Date\time of the document creation, the format can be set in the template.ini and defaults to the operating system's locate setting 
%glossary_link%
 Location to the first glossary file 
%glossary_title%
 Glossary title as set in the template.ini 
%classtree_link%
 Location to the class tree file 
%classtree_title%
 Class tree title as defined in the template.ini 
%index_link%
 Location to the overview file 
%index_title%
 The title to be used as defined in the template.ini 
%VERSION%
 The version number of UnCodeX 
%UNCODEX%
 Application title: UnCodeX 
%PLATFORM%
 The operating system used: MS Windows or GNU\Linux 
%packages_list_link%
 Location to the packages list 
%classes_list_link%
 Location to the complete classes list 
%overview_link%
 Location to the overview file 
%parentdir%
 Location to the parent dir. This is usualy zero or more times the string: "../". You can use this to create relative links to certain files. 
%targetext%
 The target extention, can be set in the template.ini file. It defaults to "html"

Template files

index.html This is the main file, it's used to create the frame sets
overview.html The main overview page, to be used as a frame on the index.html page
packages_table_entry.html A entry of a package on the main overview page
packages_list.html The list of all packages found, to be used as a frame on the index.html page
packages_list_entry.html A package entry on the packages list page
classes_list.html The list of all classes, to be used on the index.html page
classes_list_entry.html A class entry on the classes list
packageclasses_list.html The list of all classes in this package, this will be used to create the {packagename}/{packagename}-list.html files
packageclasses_list_entry.html A class entry on the package classes list
package_overview.html Per package overview page, this will be used to create the {packagename}/{packagename}-overview.html files
package_class_table_entry.html An entry of a class on the package over view
class.html Per class detail page, this will be used for every class to create the file {packagename}/{classname}.html
class_entry_const*.html class constant details
class_entry_enum*.html class enumeration details
class_entry_function*.html class function details
class_entry_state*.html class state details
class_entry_struct*.html class struct details
class_entry_variable*.html class variable details
class_entry_variable*_tag.html variable tag label
class_entry_inherited.html Details for various inherited properties
