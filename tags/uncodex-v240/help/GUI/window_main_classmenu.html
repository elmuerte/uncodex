Tree menu

Tree menu

Open

Open the selected class's source file. Or when a package has been selected, open the package directory in Microsoft Explorer.

Compile class

Compile the selected class. This will execute the command as set in the compiler commandline

Analyse class

Re-analyse the selected class.

Show properties

This will pop-up the class properties window for the selected class.

Package properties

This will pop-up the package properties window for the selected package.

Create subclass

Create a subclass of the selected class. This will display the create subclass window where you can set the values.

Delete subclass

Delete the selected class. You will be prompted if you also want to delete all subclasses. After you deleted a class you should search for references to the removed classes.

Move class

Move a class to a different package. The move class window will be shown when you can select the target package. After the move you will be prompted to search of references to the old location of the class.

Rename class

The rename class will be shown where you can rename the selected class. Only the filename and class declaration will be changed, you will still have to update all references to the old name. After the renaming you will be prompted to search for the old name.

Copy name

Copy the name of the selected package/class to the clipboard.

Output modules

This menu will only be available when output modules have been loaded (like the UnrealWikifier output modules).

Find class & Full Text Search

This will open the search dialog, it will allow you to search the tree, or the source files, for a text you enter.

Find next

This will search for the next match, this can also be used after an "inline search" done in a tree.

Expand & Collase all

This will either expand or collapse all tree nodes.
