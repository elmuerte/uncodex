HTML Template class_entry_function*.html

These replacements can only be used in class_entry_function*.html and class_entry_delegate*.html.
replacement replaced with 
%function_name%
 the name of the function 
%function_type%
 the type of function: function, event, operator, preoperator, postoperator, delegate 
%function_type_image%
 adds the location to a type image as defined in the template.ini 
%function_return%
 the return type of the function, including a link to the type definition 
%function_return_plain%
 just the return type 
%function_params%
 the function arguments including links to the type definitions 
%function_params_plain%
 just the arguments 
%function_modifiers%
 the function modifiers 
%function_state%
 the state in which this function is defined 
%function_srcline%
 the line number of the declaration 
%function_comment%
 the function description, if defined 
%function_declaration%
 syntax highlighted function declaration 
%function_init%
 link to the class where this function was first declared, this uses the text as defined by the FunctionInit title in the template.ini 
%has_comment_?%
 will be replaced with the HasCommentValue title when there's a comment 
%class_source%
 link to the class source file
