Class Properties

This will display all the details defined in a class. When you change the inheritance level it will also report details defined in parent classes.

The window will automatically disappear when it loses focus. Unless you click the little button in the upper left corner. This will transform the window so it will stay visislbe.

When you select an detail the name will be copied to the clipboard, and when you double click that item in will open the source file (uses the same command as the Full Text Search open command).
