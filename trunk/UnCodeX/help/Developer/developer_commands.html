Message commands

When UnCodeX is started with the -reuse commandline argument it will send a message to the previous instance with the new commandline arguments. You can use this same method to pass on new commands to a running UnCodeX window.

The following structure is used to pass on the new arguments: 

TRedirectStruct = packed record
  Find:      string[64];
  OpenFind:  boolean;
  OpenTags:  boolean;
  Batch:     string[255];
  NewHandle: integer;
  OpenFTS:   boolean;
end;
This structure needs to be send to the main UnCodeX window using WM_COPYDATA. 

Find
 Search for a class, leave empty if not used 
OpenFind
 Open the source file of a found class 
OpenTags
 Open the class properties window when the class is found, can not be used together with OpenFind 
Batch
 New batch commands to be executed, leave blank if not used 
NewHandle
 New window handle to send the status messages to 
OpenFTS
 Open full text search dialog
