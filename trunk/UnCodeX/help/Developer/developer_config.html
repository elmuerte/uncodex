config

Configuration file

You can use the -config commandline argument to load a diffirent configuration file. Below is the definition of the usefull config parameters.

[Config]

name type description 
AnalyseModified
 0 or 1 If set to 1 UnCodeX is search for modified files and analyse them 
ClientCmd
 commandline Commandline to execute to join the game server 
CompilerCmd
 commandline Commandline to execute to compile the selected class/package 
FullTextSearchRegExp
 0 or 1 Full text search and class body searches default to regular expression when set to 1 
HHCPath
 path Path to the HTML Help compiler (hhc.exe) 
HTMLHelpFile
 file path Output filename for the HTML Help compiler 
HTMLOutputDir
 path Path to store the output HTML files in 
OpenResultCmd
 commandline commandline to use when opening search results, if you leave this blank the default application is used 
ServerCmd
 commandline Commandline to execute when a server needs to be started 
ServerPrio
 0 to 3 Priority of the server process, 0 being the lowest and 3 being the highest 
StateFile
 filename Filename of the state file, relative to the configuration file 
TemplateDir
 path Directory which contains the templates to use for the HTML output

[HotKeys]

This is where the horkeys are set for each menu/toolbar item.

[IgnorePackages]

Package
 Directories matching these names will be ignores if they are identified as a package.

[Layout]

name type description 
ABLeft
 0 or 1 Is registered as a tool application to the left side 
ABRight
 0 or 1 Is registered as a tool application to the right side 
AutoHide
 0 or 1 The tool application is set in autohide mode 
ABWidth
 integer Width of the tool application window 
ExpandObject
 0 or 1 Expand the first item in the class tree on startup 
Height
 integer The height of the normal UnCodeX window 
Left
 integer Position to the left of the main window 
Log
 0 or 1 Show log window if set to 1 
Log.Color
 color Background color of the log window 
Log.Font.Color
 color Text color 
Log.Font.Name
 name Font to use in the log window 
Log.Font.Size
 integer Font size 
LogHeight
 integer Height of the log window 
MenuBar
 0 or 1 Show the menu bar if set to 1 
MinimizeOnClose
 0 or 1 Minimize UnCodeX when the user closes the window, unless it's closed via "Tree -> Quit" 
PackageTree
 0 or 1 Show the package tree 
PackageTreeWidth
 integer Width of the package tree 
SavePosition
 0 or 1 Save the position of UnCodeX 
SaveSize
 0 or 1 Save the size of UnCodeX 
StayOnTop
 0 or 1 If set to 1 the main window willstay on top of all other windows 
Toolbar
 0 or 1 Show the toolbar 
Top
 integer Position to the top of the main window 
Top
 integer Position to the top of the main window 
Tree.Color
 color Background color of both trees 
Tree.Font.Color
 color Text color 
Tree.Font.Name
 name Font to use in both trees 
Tree.Font.Size
 integer Font size 
Width
 integer Width of the main window

[PackagePriority]

Package
 The priority of the package, the first item has the highest priority. 
Tag
 A tagged package, this package and containing classes will get a diffirent icon in the tree listing. The order doesn't matter

[SourcePaths]

Path
 Paths to scan for packages
