Commandline options

UnCodeX accepts the following commandline options: 
-batch command ...
 Start UnCodeX in batch mode, command can be one or more of the following seperated by spaces: 
rebuild
 rebuilds the class tree 
analyse
 analyse all classes in the tree 
createhtml
 create all HTML file 
htmlhelp
 compile the MS HTML Help file 
close
 close UnCodeX 
ext:<name>
 call the custom output module out_name.dll 
--
 end of batch command input, use this if you want to add more command line options after the batch sequence 
-config filename
 Use filename as config file 
-find class name
 Find and select class name in the class tree 
-handle window handle
 The window with window handle will receive status messages from UnCodeX, read more about this option in the developers section 
-help
 Displays a list of accepted commandline options 
-hide
 don't show the main window of uncodex, note that this will prevent you from shutting it down when you don't use "close" the batch command 
-open class name
 Find and open the source file of class name 
-reuse
 Re use a previous UnCodeX instance, this option depends on the configuration file used 
-tags class name
 Find and open the details of class name 
-fts
 Show the Full Text Search dialog on startup
