These replacements can only be used in class_entry_variable*.html, struct_properties_detail*.html and in struct_properties_comment*.html.
replacement replaced with 
%var_name%
 the name of the variable 
%var_type%
 the type of the variable, including a link to the type declaration 
%var_type_plain%
 just the type name 
%var_modifiers%
 the modifiers of this variable 
%var_tag%
 the config name tag, e.g. "var(tag) type varname" 
%var_srcline%
 the line number of the declaration 
%var_comment%
 description of the variable 
%var_declaration%
 syntax hilighted declaration of the variable 
%has_comment_?%
 will be replaced with the HasCommentValue title when there's a comment 
%class_source%
 link to the class source file
