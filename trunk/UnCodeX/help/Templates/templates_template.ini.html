template.ini

HTML Templates template.ini

This file configures the template used.

[Titles]

Overview=
 the title for the overview link 
Glossary=
 the title for the glossary link 
ClassTree=
 the title for the class tree link 
DirectSubclasses=
 direct subclasses of the class 
Constants=
 used as title for the class constants 
Variables=
 used as title for the class variables 
Enums=
 used as the title for the class enumerations 
Structs=
 used as the title for the class structures 
Functions=
 used as the title for the class function 
InheritedConstants=
 used as title for inherited constants 
InheritedVariables=
 used as title for inherited variables 
InheritedEnums=
 used as the title for inherited enumerations 
InheritedStructs=
 used as the title for inherited structures 
InheritedFunctions=
 used as the title for inherited function 
DefaultTitle=
 default title replacement

[CopyFiles]

File=
 file to copy to the destination directory, use this to copy images and stylesheets to the target directory

[Settings]

MaxInherit=
 maximum parent classes to include in the inherited const/vars/etc. listing 
CreateGlossary=
 create the glossary if set to 1 
CreateClassTree=
 create the class tree if set to 1 
SourceCode=
 create syntax hilighted source code if set to 1 
TargetExt=
 The target extention, this can be overwritten from within UnCodeX 
TabsToSpaces=
 Convert tabs to # spaces, when lower or equal to 0 don't convert 
CPP=
 Comment PreProcessor to use (relative to the Template directory)
