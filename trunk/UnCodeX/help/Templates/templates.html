HTML Templates

HTML Templates consist out of HTML formatting and replacements. Replacements are depended on the template file.

General replacements

These replacements can be used in any template.
replacement replaced with 
%default_title%
 DefaultTitle as set in the template.ini, defaults to "UnCodeX" 
%create_time%
 Date\time of the document creation, format is defined by the operating system's locate setting 
%glossary_link%
 Filename of the glossary: "glossary_A.html" 
%glossary_title%
 Glossary title as set in the template.ini 
%classtree_link%
 Filename of the class tree page: "classtree.html" 
%classtree_title%
 Class tree title as defined in the template.ini 
%index_link%
 Filename of the main overview page: "overview.html" 
%index_title%
 The title to be used as defined in the template.ini 
%VERSION%
 The version number of UnCodeX 
%UNCODEX%
 Application title: UnCodeX

Template files

index.html This is the main file, it's used to create the frame sets
packages_list.html The list of all packages found, to be used as a frame on the index.html page
packages_list_entry.html A package entry on the packages list page
classes_list.html The list of all classes, to be used on the index.html page
classes_list_entry.html A class entry on the classes list
overview.html The main overview page, to be used as a frame on the index.html page
classtree.html The template for the class tree
glossary.html The template for the class tree
packages_table_entry.html A entry of a package on the main overview page
package_overview.html Per package overview page, this will be used to create the package_{packagename}.html files
package_class_table_entry.html An entry of a class on the package over view
class.html Per class detail page, this will be used for every class to create the file {packagename}_{classname}.html
class_const_entry.html An entry for a class constant
class_enum_entry.html An entry for a class enumeration
class_function_entry.html An entry for a class function/event/operator
class_struct_entry.html An entry for a class structure
class_var_entry.html An entry for a class variable
class_inherited.html Used to display inherited variables/functions/etc.
sourcecode-#.html The top and bottom of the syntax hilighted source code
